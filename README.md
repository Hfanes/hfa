## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Hfanes/hfa.git
```

2. Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Run

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Website runs at: http://localhost:3000
